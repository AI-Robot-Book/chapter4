# 地図ファイル

## 概要
４章のナビゲーション用の地図ファイル．
この地図ファイルhouse.pgmとhouse.yamlは参考文献を改変したものです．

## インストール
書籍の説明ではmapディレクトリは$HOME/mapのパスになっているので次のコマンドでファイルをコピーします．
```
cp -r ~/airobot_ws/src/chapter4/src/map ~
```
